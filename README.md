# Sornitool-Free
## This tool can be Open repak pubgm or peace elite.

## Termux installation

```
pkg update && pkg upgrade 
```
```
pkg install git
```
```
termux-setup-storage
```
## Clone the image repository
```
git clone https://github.com/oeint/OR.git
```
```
cd OR
```
```
chmod +x setup
```
```
./setup
```
## Shortcut command
```
OR
```
```
OR --Install
```
```
OR --settings
```
```
OR --help
```
